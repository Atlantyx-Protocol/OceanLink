# References

This is the list of papers, reports, and sources I used in the thesis. The list
comes from the thesis bibliography
([`docs/thesis/references.bib`](../docs/thesis/references.bib)) and uses the same
groups. When you change one, change the other too so they stay the same.

> **PDFs:** Put a PDF copy of each source in this folder, and name it after its
> citation key (for example `notland2024sok.pdf` or `poon2016lightning.pdf`).
> Some sources are blog posts or web docs, not real PDF files. For those, the
> link is the main source, but you can still save the page as a PDF (with the
> date) to keep a copy.
